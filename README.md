# 💼 Personal Portfolio Website

A responsive personal portfolio website to showcase my skills, projects, experience, and contact information.

---

## 🚀 Features

- Fully responsive design (mobile + desktop)
- Smooth navigation between sections
- Projects showcase with GitHub and live links
- Contact section with social media integration
- Clean and modern UI design

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript  

---

## 📁 Project Structure
/index.html -> Main structure
/style.css -> Styling and layout
/script.js -> Interactivity (if used)
/assets -> Images and icons

---

## 🎯 Purpose

This portfolio was built to:
- Showcase my technical skills and projects  
- Provide easy access to my GitHub and LinkedIn profiles  
- Serve as a personal branding website for recruiters
